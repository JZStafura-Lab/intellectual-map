# Intellectual Network Map

Interactive visualization of 450+ books organized by core problems (consciousness, violence, social structure, aesthetics, etc.). Shows connections between domains, identifies gaps, and maps key authors who bridge multiple fields.

## Features
- 8 intellectual problem domains with connection strengths
- Key authors/works as bridge nodes
- Interactive exploration (zoom, pan, hover for details)
- Node sizing based on reading volume

## Quick Start
```bash
pip install -r requirements.txt
python src/create_network_viz.py
open visualizations/intellectual_network_map.html
